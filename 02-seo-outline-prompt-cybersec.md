# Prompt 2: The SEO Blog Outline Generator (B2B Cyber Security)

**Purpose:** Use this prompt to create a structurally sound, E-E-A-T optimized outline (H1-H3) that addresses B2B buyer pain points and compliance regulations.

## The Prompt

> **Act as an Expert B2B SEO Content Manager.** > 
> Create a comprehensive, well-structured blog outline for the following title: **"[Insert Blog Title Here]"**.
> 
> **Requirements for the Outline:**
> - **H1 Tag:** Create a compelling, SEO-optimized H1 that speaks to a business owner or IT Director's pain points.
> - **Subheadings (H2 & H3):** Provide logically structured H2 and H3 subheadings. Ensure you include sections that touch upon compliance (e.g., SOC 2, HIPAA, GDPR) and risk mitigation.
> - **Local/Industry SEO Placements:** Identify 2-3 specific spots in the outline where I can naturally include the target keyword: **"[Insert Service + City/Industry, e.g., Phishing Simulations for Austin Tech Startups]"**.
> - **PAA Section:** Include a section for "Frequently Asked Questions" that targets Google's 'People Also Ask' queries related to this topic (e.g., "How often should employees undergo security training?").
> - **Internal Links:** Suggest 2 internal links to other relevant topics (e.g., link to the Phishing Simulator Product page or the Compliance Guide).
> - **Call to Action (CTA):** End with an outline for a strong, B2B-focused CTA encouraging the reader to book a demo, download a security checklist, or request a risk assessment.

## How to Use:
Run this prompt for the Pillar Blog and each Supporting Blog individually. Wait for the AI to output the outline, review it, and then proceed to the content generation step.
