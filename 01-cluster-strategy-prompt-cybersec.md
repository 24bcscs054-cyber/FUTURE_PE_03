# Prompt 1: The Content Cluster Strategy Generator (B2B Cyber Security)

**Purpose:** Use this prompt to generate a highly relevant, localized, and compliance-focused SEO content cluster for a B2B cybersecurity firm.

## The Prompt

> **Act as a Senior SEO Strategist for a B2B SaaS and IT Services agency.** > 
> I am building a content cluster for a **[Insert Business Type, e.g., Cyber Security Awareness Firm]** located in **[Insert City/Region, e.g., Austin, Texas]**. The main service we want to promote and rank for is **[Insert Main Service, e.g., Corporate Phishing Simulations and Employee Security Training]**. 
> 
> Please generate a strategic content cluster mapping out the following:
> 
> 1. **One Pillar Blog Title:** This must be a broad, comprehensive guide targeting the main service + target audience/city (e.g., "The Complete Guide to Employee Cyber Security Training in [City]").
> 2. **Four Supporting Blog Titles:** These should target specific B2B pain points, such as phishing simulation metrics, SOC 2 / HIPAA compliance requirements, the cost of data breaches, or remote worker security.
> 3. **Search Intent:** Identify the primary search intent (Informational, Commercial Investigation, Transactional, or Local Navigational) for each title.
> 4. **Target Keywords:** Provide 2-3 long-tail or LSI keywords for each title (e.g., "how to prevent spear phishing," "security awareness training for remote teams").

## How to Use:
Replace the bracketed information `[ ]` with your specific client details before pasting into your preferred AI tool.
