# 🛡️ B2B Cyber Security SEO Content Cluster System

## 📌 Project Overview
This repository contains a complete, automated prompt engineering system designed to generate high-ranking, locally optimized SEO content clusters for service-based businesses. 

Rather than generating random, unstructured AI articles, this system uses a strategic **Pillar and Cluster** model. It generates structured outlines (H1-H3), defines search intent, and enforces strict E-E-A-T (Experience, Expertise, Authoritativeness, and Trustworthiness) guidelines, which is critical for B2B and YMYL (Your Money or Your Life) niches like cybersecurity.

## 🏢 Use Case Context
To demonstrate the system's capabilities, the prompts were executed for a specific local business scenario:
* **Business Type:** B2B Cyber Security Awareness & Training Firm
* **Target Location:** Gandhinagar & GIFT City, Gujarat, India
* **Primary Goal:** Drive local leads from IT Directors and Business Owners looking for compliance and phishing simulation training.

## 🗺️ The Content Strategy
The system generated the following SEO content map:

| Content Type | Topic / Target Keyword | Search Intent |
| :--- | :--- | :--- |
| **Pillar Blog** | The Complete Guide to Employee Cyber Security Training in Gandhinagar | Informational / Trust-building |
| **Supporting 1** | Why Phishing Simulations are Critical for Tech Companies in GIFT City | Commercial Investigation |
| **Supporting 2** | The True Cost of Data Breaches for Businesses in Gujarat | Informational |
| **Supporting 3** | Top 5 Social Engineering Tactics Targeting IT Firms in 2024 | Informational |
| **Supporting 4** | How to Choose the Best Cyber Security Training Provider in Gandhinagar | Local Navigational |

## 📁 Repository Structure
* `/prompts/` - Contains the 3 reusable prompt templates (Strategy, Outline, Content Writer) that can be adapted for any client or industry.
* `/Main Content-pillar-blog-cybersecurity-gandhingar/` - Contains the final, AI-generated markdown blogs for the Gandhinagar Cybersecurity case study, complete with SEO-optimized headers, local context, and internal linking structures.

## ⚙️ How to Use the Prompts
To use this system for your own agency or business:
1. Open the files in the `/prompts` folder.
2. Copy the text and replace the bracketed information `[like this]` with your specific business, city, and target service.
3. Run the prompts sequentially in ChatGPT, Claude, or Google Gemini.
